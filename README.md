## 🚀 Проекты

### 1. 📈 Когортный анализ Retention мобильного приложения (Продуктовая аналитика)
**Цель:** Оценить удержание пользователей и выявить факторы, влияющие на возвращаемость.
**Инструменты:** Python, Pandas (groupby, pivot, merge), Seaborn (heatmap).
**Ключевые инсайты:**
- Retention 30-го дня вырос с 20% (сентябрь) до 33% (ноябрь), что говорит о приросте лояльности на 65%.
- Выдвинуты гипотезы о влиянии сезонности и изменений в онбординге на качество когорт.
**Файлы:** `cohort_retention_analysis.ipynb`, `retention_heatmap.png`

### 2. 💼 Анализ рынка вакансий Data Analyst (HR-аналитика)
**Цель:** Проанализировать требования рынка труда, зарплатные вилки и ключевые навыки для начинающих специалистов.
**Инструменты:** Python, Pandas (очистка данных, fillna, агрегация), Matplotlib.
**Ключевые инсайты:**
- SQL и Python являются обязательным минимумом (встречаются в ~100% вакансий).
- Удаленный формат работы является оптимальной стратегией для соискателей из регионов, сочетая высокие зарплаты с низкой стоимостью жизни.
**Файлы:** `vacancy_analysis.ipynb`

### 3. 📚 Анализ рынка книг: парсинг и EDA (Web Scraping)
**Цель:** Собрать данные с сайта books.toscrape.com и проанализировать зависимость цены от рейтинга.
**Инструменты:** Python, requests, BeautifulSoup, Pandas, Matplotlib, Seaborn.
**Ключевые инсайты:**
- Обнаружен контринтуитивный паттерн: книги с максимальным рейтингом (5) в среднем стоят дешевле, чем книги с рейтингом 1.
- Гипотеза: цена определяется не рейтингом, а другими факторами (жанр, автор, формат).
- **Рекомендация:** не ориентироваться только на рейтинг при прогнозировании цены.
**Файлы:** `books_scraping_analysis.ipynb`, `price_vs_rating.png`
