# 📊 Анализ рынка вакансий Data Analyst

## 🎯 Цель проекта
Проанализировать актуальные требования рынка труда к Data Analyst, чтобы выявить ключевые навыки, географические особенности и реальный уровень заработных плат. Проект помогает сформировать стратегию обучения и поиска работы для начинающих специалистов.

## 🛠 Используемые инструменты
- **Python**: Pandas (очистка, группировка, агрегация), Matplotlib (визуализация)
- **Статистика**: Анализ медианы vs среднего, работа с выбросами и пропусками
- **Среда**: Google Colab / Jupyter Notebook

## 💡 Ключевые бизнес-инсайты

1. **Зарплатная воронка нелинейна**: Переход от Junior к Middle дает прирост медианной нижней границы зарплаты на ~50%. Однако ключевой скачок происходит при переходе к уровню Senior (+300% от Junior). Первый год опыта критически важен для закладки фундамента дохода.
2. **Обязательный стек навыков**: SQL и Python встречаются практически в 100% вакансий, Excel — в 50%. Остальные инструменты (Tableau, Power BI, A/B тесты) являются дифференциаторами. *Рекомендация:* Достаточно выбрать один BI-инструмент и изучить его на базовом уровне, чтобы закрыть большинство требований.
3. **Географический парадокс**: Москва лидирует по количеству вакансий, но уступает Санкт-Петербургу по уровню зарплат. **Удаленный формат работы** оказывается оптимальной стратегией: он сочетает высокий уровень оплаты с доступом ко всему рынку и низкой стоимостью проживания в регионах.

## 📂 Структура проекта
- `vacancy_analysis.ipynb` — основной Jupyter Notebook с кодом, визуализациями и комментариями.

## 🚀 Как запустить
1. Клонируйте репозиторий: `git clone <ссылка_на_репозиторий>`
2. Откройте файл `vacancy_analysis.ipynb` в Jupyter Notebook или Google Colab.
3. Запустите все ячейки последовательно.
